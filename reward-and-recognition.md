# 獎勵機制

本機制適用於 **第二十二梯所有學員**。

彈性與福利是建立在「穩定交付」與「互信」的基礎之上；因此透過具體成果的完成，逐步累積教練對你的信任，進而換取更多彈性空間。

相關概念也可以運用到未來職場或在職工作的情境中。

　

## 1️⃣ 適用對象與原則

- 獎勵建立在「如期完成既定目標，且符合基本品質要求」之上，並以教練認定為準。
- 各項獎勵為「個人獎勵」，以個人實際完成情況核發，不採團體平均。

　

### **JS 組**

| 項目 | 獎勵內容 |
|------------------------|-------------|
| 完成「甜點電商」           | 早退一小時    |
| 完成「1-1 對談（第三個月）」 | 額外一天調適假 |
| 完成「天氣 App」          | 晚到一小時    |

　

### **C# 組 & UI 組**

| 項目 | 獎勵內容 |
|--------------|-------------|
| 完成「甜點電商」 | 早退一小時    |
| 培訓第三個月   | 額外一天調適假 |
| 培訓第五個月   | 晚到一小時    |

　

## 2️⃣ 獎勵使用規範

### 1. **早退一小時 / 晚到一小時**

**使用方式**：
- 須在 `Discord: 📮｜培訓簽到` 頻道的簽入、簽出訊息標示。
  ```
  09:50 阿劍 簽入（晚到一小時）
  ```
  ```
  17:00 阿劍 簽出（早退一小時）
  ```
- 不得與「🔑 鑰匙保管人」的時間衝突：
  - 若當日同時擔任鑰匙保管人，需先完成交接或調整排班，始得使用。
- 不得使用於 **重要活動日**（例：考試、專題發表等）。
- 原則上 **同一日僅可擇一使用**，即「不可」當日晚到又早退。

　

### 2. **調適假**

- 申請流程比照一般「調適假」，於 `Discord: 😈｜請假回報` 頻道說明。
  ```
  12/01（一）22梯 阿劍 調適假（額外）。
  ```
- 請假紀錄中會標註為「獎勵調適假」，以利後續統計與檢視。

　

## 3️⃣ 注意事項

- 獎勵目的在於 **強化「信任來自穩定交付」的觀念**，而非鼓勵過度消耗出勤時數。
- 若學員同時發生以下情況之一，教練可以 **暫緩** 或 **取消** 其獎勵的使用權：
  - 出缺勤違規次數偏高，已影響學習進度或團隊協作。  
  - 正在啟動或進行 [退訓機制](/withdrawal-and-termination.md) 評估流程。  
  - 有其他明顯違反 [生活公約](/code-of-conduct.md) 之行為。
