# 出缺勤與請假規範

本規範適用於 **第二十二梯所有學員**，旨在維持日常作息與學習品質。

相關習慣與流程，也可以運用到未來職場或在職工作之出勤管理。

　

## 1️⃣ 每日簽到流程

### 🔑 **鑰匙保管人**

- 可固定同一人，或每日（週）輪流擔任。
- 週一至週五須在 **08:30–08:50** 抵達基地並負責開門。
- 抵達後，在 `Discord: 📮｜培訓簽到` 頻道建立當日討論串：`yyyy/mm/dd`。
  ```
  2025/12/01
  ```
- **18:00 以後**，在當日同一討論串中回覆：`hh:mm 學員 簽出`。
  ```
  18:00 阿劍 簽出
  ```

⚠️ 超過 08:50 才建立當日討論串，視作「遲到」。

　

### 🙍 **學員**

- 週一至週五須在 **09:00 以前** 抵達基地。
- 抵達後，在 `Discord: 📮｜培訓簽到` 頻道當日討論串中回覆：`hh:mm 學員 簽入`。
  ```
  08:50 阿劍 簽入
  ```
- **18:00 以後**，在當日同一討論串中回覆：`hh:mm 學員 簽出`。
  ```
  18:00 阿劍 簽出
  ```

⚠️ 在 09:01-09:10 才簽入，視作「遲到」。

　

## 2️⃣ 請假流程

### 🔑 **鑰匙保管人**

- 須在 **請假前一日 18:00 以前**，指派隔日的鑰匙保管人，取得對方同意後交接鑰匙。
- 須在 **請假當日 09:00 以前**，在 `Discord: 😈｜請假回報` 頻道告知假別與 **指派的鑰匙保管人**。
  ```
  12/01（一）22梯 阿劍 調適假，當天的鑰匙保管人已請 @Ayu 擔任。
  ```
- 如遇週末無法交接鑰匙，或臨時需要請假，請儘速聯繫工作人員（Ayu），協助當日開門及討論串的建立。

　

### 🙍 **學員**

- 須在 **當日 09:10 以前**，在 `Discord: 😈｜請假回報` 頻道說明假別。
  ```
  12/01（一）22梯 阿劍 調適假。
  ```

　

⚠️ 未能在指定時間以前完成請假，一律視作「缺席」。

　

## 3️⃣ 假別說明

### 1. **調適假**
  - 一週最多申請 **一天**。
  - 於 `Discord: 😈｜請假回報` 頻道告知「調適假」即可。
  ```
  12/01（一）22梯 阿劍 調適假。
  ```

### 2. **病假**
  - 一個月最多申請 **三天**。
  - 於 `Discord: 😈｜請假回報` 頻道告知「病假」，並簡要說明狀況。
  ```
  12/01（一）22梯 阿劍 病假（發燒）。
  ```

### 3. **事假**
  - 一個月最多申請 **三天**（視下方「5️⃣ 特殊狀況」，得由教練斟酌增減）。
  - 須先在 Discord 私訊教練，說明請假原因，並 **取得教練的文字同意**。
  - 取得同意後，在 `Discord: 😈｜請假回報` 頻道告知「事假」。
  ```
  12/01（一）22梯 阿劍 事假。
  ```

　

⚠️ 未依照上述假別流程請假，一律視作「違規」。

　

## 4️⃣ 違規與退訓

- 「遲到」、「缺席」以及「違規」等情形，若累積達一定次數，火箭隊將依照 [退訓機制](/withdrawal-and-termination.md) 進行退訓評估與處理。

　

## 5️⃣ 特殊狀況

- 如遇 **教招**、**生理因素**、**出國旅遊**、**身心狀態**、**生活重大變故** 等特殊情況，請儘早向教練與工作人員（Ayu）說明狀況，由相關人員協助評估出缺勤認定與後續處理方式。
- 若狀況可能影響長期出勤，建議主動討論可行的調整方案，以兼顧個人身心狀態與培訓進度。
